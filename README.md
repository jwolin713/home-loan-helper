# Home Loan Helper

[![CI](https://github.com/jwolin713/home-loan-helper/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/jwolin713/home-loan-helper/actions/workflows/ci.yml)

A web app to help first-time homebuyers understand affordability using real financial data and live mortgage rates.

- Development plan: `docs/DEVELOPMENT_PLAN.md`
- PRs welcome; CI runs Markdown lint and Node checks when applicable.

## Local Development

- Clone and install deps (if present): `npm install`
- Run tests (if defined): `npm test`
- Lint (if defined): `npm run lint`
